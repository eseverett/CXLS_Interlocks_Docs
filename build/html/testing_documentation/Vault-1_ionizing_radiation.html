Vault-1 Ionzing Radiation Interlock Testing Protocol

The objective of this testing procedure is to verify the funcationality of the Vault-1 ionizing radiation interlock system. The following fall under the scope of this testing procedure: - Vault-1 search and securing system. - RF and accelerator arming. - Tranmitter override. - Ionzing radiation monitoring interlocks.

Starting Conditions

The default state for testing of the Vaut-1 ionizing radiation interlock system is in a completely disarmed state.

  1. The IONIZING RADIATION INTERLOCK protocase in Vault-1 Control shows:

    • All SECURE PERIMETER lamps are red.

    • All AREA MONITOR lamps are green.

    • TRANSMITTERS and ACCERATOR lamps are red.

  2. VIEWMARQ in Vault-1 Control displays LASER SAFE.

  3. VIEWMARQ in Accererator Lab disaplys RF SAFE.

  4. All individual red, blue, and orange beacons are off.

  5. Search buttons LEDs are off in Vault-1.

    • Button 1

    • Button 2

    • Button 3

Testing Unsecure Vault-1 Conditions

  1. Switch the ENABLE key for the accelerator. The accelerator lamp remains red.

  2. Swith the ENABLE keys for transmitters 1 and 2. Both transmitter lamps remain red.

Searching Procedure

  1. Push search buttons three and two and verify they will not activate without going in the correct sequence.

    • Button 3

    • Button 2

  2. Go through the vault and in sequence, click the three search buttons. The LED on the search button should glow and the corresponding lamp on the Vault-1 Control IONIZING RAIDATION INTERLOCK protocase should turn green.

    • Button 1

    • Button 2

    • Button 3

  3. When the third search button is hit, a chime should be audible.

  4. Once all three search buttons have been hit in order, close the shield door. The shield door lamp on the Vault-1 Control IONZING RADIATION INTERLOCK protocase will turn green once the mechanical door switches are fully actuated.

Arming the Accererator and transmitters

  1. With the shield door still closed, enable the accelerator on the Vault-1 Control IONIZING RADIATION INTERLOCK protocase using the ENABLE key. The accelerator status lamp should turn green.

  2. Turn the ENABLE key for transmitter 1 on the Vault-1 Control IONIZING RADIATION INTERLOCK protocase. The transmitter 1 status lamp should turn green.

  3. Once the transmitter is enabled, the VIEWMARQ diaplys will show VAULT SECURE - RF ARMED.

    • Vault-1 Control

    • Accererator Lab

  4. The blue beacons next to each VIEWMARQs are on.

    • Vault-1 Control

    • Accererator Lab

  5. Hit the reset button and repeat 2-4 with transmitter 2.

Warning

NEEDS VIEMARQ IMAGES

Overriding and Reseting Transmitters and Accererator

  1. With the accelerator and transmitters aremed, swithc the OVERRIDE keys on the Vault-1 Control IONIZING RADIATION INTERLOCK protocase. The TRANSMITTERS lamps will turn orange.

  2. Switch the OVERRIDE keys back to interlock. The TRANSMITTERS lamps will turn green.

  3. Hit the reset button on the Vault-1 Control IONIZING RADIATION INTERLOCK protocase. The ACCELERATOR and TRANSMITTERS lamps will turn red.

Timing out the Shield Door and Search sequence

  1. Rearm the accelerator and transmitters and open the shield door. The accelerator and transmitter lamps should turn red.

  2. Push only the first search button. After x seconds, the button LED should turn off.

  3. Go through the search procedure again except do not close the shield door and allow the system to trip. After x, the search lamps should turn red.

Return to Starting Conditions

  1. Return Vault-1 ionizing radiation interlock system to the default state.
